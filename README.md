# Remix Typescript App Template

## Stack

- Typescript (client, server)
- ESLint
- Prettier
- husky
- lint-staged
- gitmoji
- concurrently
- nodemon

## How to use

- Install Package

```
$ yarn
```

- Setup vscode

```
$ cp -r .vscode.sample .vscode
```

- Start Development!

```
$ yarn dev
```

## commit

```
$ git commit
```
